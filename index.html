Projects

LatteScript

LatteScript is an educational programming language and environment for use in web browsers. You can try it out on GitHub. I have blogged extensively about its development at the official LatteScript blog. The entire source code is available on GitHub.

pypixel

I learned programming using the graphics routines in QBasic. They were extremely basic, but perfect for a beginning programmer. I have instructed many people in the use of QBasic graphics, and grown fond of it. I think Python is one of the best programming languages for education right now, and that it needed a library that covered simple graphics as in QBasic. I spoke about this project at the SuperQuest Spring Conference in 2011. You can watch a video of my presentation on Vimeo.

npcworld

npcworld is an organism simulator. It simulates basic organism in a world, and also simulates a universe of worlds, with migration or organisms from one world to another. It is configurable and internally documented. You can download the code here.

visint

visint is a volume integral visualizer. I created this as a visual learning aid for students in multivariable calculus. It is capable of visualizing integration of three different functions (but would be easily extendable to more) and allows the user to fly through the space or rotate/pan around with mouse controls. It was written in Java using pure OpenGL.

mprw

mprw is a simple web interface for controlling an mpd server. mpd is a program which plays music and organizes music libraries, with many different interfaces. Many web interfaces already existed before I wrote this, but they did not meet my requirements of simplicity, beauty, and ease-of-use. I have not made a stable release of mprw yet, but it is completely usable in its current state. mprw makes use of JavaScript (including jQuery) to run asynchronously without page refreshes, and is optimized for Android phones (and laptops/desktops, of course).

gridpy

Gridpy is a simple grid-based graphics library for Python that I made in order to implement Conway's game of life. I thought it was general enough to release to the public, so I put it on GitHub.

8-beat

This project is a music creation program that uses the PC speaker to make retro style music. It runs on Linux and Windows, and is well documented. If you visit the Google Code page, it has a link to the SourceForge page which hosts the HTML documentation for the program.

I worked closely with my musically inclined friends to produce a simple and useful program, since I know little about music myself.

I provided extensive documentation for learning how to use the software I wrote, in hopes of enabling musicians to make interesting music using my software.
